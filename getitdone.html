Get-It-Done!

A Reminder App

Project Date: 12/25/2018 - Present

Previous Next

Get-It-Done is a personalized reminder Andriod application. As a registered user, it protects the privacy of your schedule with a numeric password. All the data is stored in a local SQLite Database. You can set the time of the notification, and your phone will alarm you at the inputted time.

Andriod Features Implemented

  • SQLite Database
  • Recycler View
  • Card View
  • Notification
  • Broadcast Receiver

Design Pattern: Model View Controller
